== Docker Compose

@see `.env` for shared environment variables.

[source,bash]
....
docker-compose up -d
....

== Debug

In `devenv/`

[source,bash]
....
docker build init-keycloak -t devenv/keycloak
docker run -v $(pwd)/init-keycloak:/scripts --entrypoint /bin/bash --network bff_default --env-file .env -it devenv/keycloak
....
